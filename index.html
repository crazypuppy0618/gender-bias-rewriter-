E · 现状对比 Current vs Solution

❌ Current Approach 现状

  1. 用户输入 "女人太情绪化"
  2. 关键词过滤(无脏字 → 通过)
  3. "未发现违规"
  4. 帖子发布 → 敌意与伤害

✅ Our Solution 本方案

  1. 用户输入 "女人太情绪化"
  2. 改写提示检测到隐式偏置
  3. 建议中性表达 "人的情绪因人而异"
  4. 用户接受 → 中性发布 → 伤害减少

优势 Advantages

  • ✓ 预防胜于删除:在内容扩散前介入
  • ✓ 教育而非惩罚:促进长期行为改变
  • ✓ 符合监管:对齐中国"清朗"专项与 SDG 5
  • ✓ 试点有效:职场研究显示偏置语言 >30% 下降且无用户抵触
